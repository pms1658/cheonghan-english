# 🔥 3분 만에 끝내는 '서버 만들기' 가이드

코딩을 전혀 몰라도 괜찮습니다! 인터넷 회원가입하듯이 마우스 클릭만으로 따라와 주세요.

---

## 1단계: 구글 로그인하고 사이트 접속
1. 이 주소를 클릭하세요 👉 [https://console.firebase.google.com/](https://console.firebase.google.com/)
2. 구글 아이디로 로그인하세요. (평소 쓰시는 Gmail이면 됩니다)
3. 화면 중간에 **[프로젝트 만들기]** (또는 **+ 프로젝트 추가**) 버튼을 누르세요.

## 2단계: 프로젝트 이름 짓기
1. 프로젝트 이름 칸에 `cheonghan-english` 라고 적으세요. (아무거나 상관없어요)
2. **[계속]** 버튼 클릭.
3. "Google 애널리틱스" 화면이 나오면? **[사용 설정 위젯 끄기 (비활성화)]** 하고 **[프로젝트 만들기]** 클릭.
   - *복잡한 기능은 지금 필요 없어요!*
4. 잠시 기다리면 "새 프로젝트가 준비되었습니다"라고 뜹니다. **[계속]** 클릭.

## 3단계: 비밀번호 로그인 켜기
1. 왼쪽 메뉴에서 **[빌드]** 라는 글자를 찾아 누르고, 그 아래 **[Authentication]**을 클릭하세요.
2. 화면 가운데 **[시작하기]** 버튼 클릭.
3. **[이메일/비밀번호]** 라고 적힌 부분을 클릭.
4. **[사용 설정]** 스위치를 켜고 **[저장]** 클릭.
5. 이제 **[Users]** 탭을 클릭하고 **[사용자 추가]** 버튼을 누르세요.
   - 이메일: `cheonghan_eng@cheonghan.com`
   - 비밀번호: `cjdgks87`
   - **[사용자 추가]** 완료! (이게 선생님 관리자 아이디가 됩니다)

## 4단계: 데이터베이스 만들기 (저장소)
1. 왼쪽 메뉴에서 **[빌드]** > **[Firestore Database]** 클릭.
2. **[데이터베이스 만들기]** 버튼 클릭.
3. 위치는 `Seoul` (또는 `asia-northeast3`) 선택하면 좋고, 못 찾겠으면 그냥 미국(`us-central1`) 그대로 두고 **[다음]** 클릭.
4. **중요!** **"테스트 모드에서 시작"** 을 선택하고 **[만들기]** 클릭. (프로덕션 모드 아님!)

## 4.5단계: 파일 저장소 만들기 (TTS 음성 캐싱용)
> 이미 프로젝트가 있으면 해당 프로젝트에서 아래만 진행하세요!
1. 왼쪽 메뉴에서 **[빌드]** > **[Storage]** 클릭.
2. **[시작하기]** 버튼 클릭.
3. **"테스트 모드에서 시작"** 선택하고 **[다음]** 클릭.
4. 위치는 그대로 두고 **[완료]** 클릭.
5. 끝! TTS 음성 파일 저장이 가능해집니다.

## 5단계: 키 가져오기 (마지막!)
1. 화면 왼쪽 맨 위에 **[톱니바퀴 아이콘 ⚙️]** 을 누르고 **[프로젝트 설정]** 클릭.
2. 화면을 맨 아래로 쭈욱 내리세요. "내 앱" 이라는 곳이 보입니다.
3. **[</>]** (꺾쇠 모양 아이콘)을 클릭하세요.
4. 앱 닉네임에 `my-app` 이라고 적고 **[앱 등록]** 클릭.
5. 그러면 영어가 잔뜩 나옵니다. `const firebaseConfig = { ... }` 이 부분을 찾으세요.
6. 그 안에 있는 내용들(`apiKey`, `authDomain` 등등)이 필요합니다.
